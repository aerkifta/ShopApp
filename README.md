## Описание проекта

В рамках контрольной работы было разработано Android-приложение «Магазин».

Приложение предназначено для хранения и обработки информации о клиентах, товарах и заказах.

В приложении реализованы:
- работа с объектами «Клиент», «Заказ» и «Продукт»;
- отображение информации о заказах;
- подсчет стоимости заказов;
- вывод популярных товаров;
- поиск товаров по заданным параметрам;
- пользовательский интерфейс Android-приложения;
- работа с компонентами Android SDK.

Также реализованы функции:
- получение списка заказов клиента за указанный период;
- вывод 5 наиболее популярных товаров за прошлый месяц;
- фильтрация товаров по названию и максимальной цене.

## Тестирование

Тестирование и отладка приложения проводились на физическом смартфоне `Xiaomi 12`.

Подключение устройства осуществлялось по Wi-Fi через Android Debug Bridge (ADB).

Использование физического устройства было выбрано из-за высокого потребления оперативной памяти встроенным эмулятором Android Studio.
